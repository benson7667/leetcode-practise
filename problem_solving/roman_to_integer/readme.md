## Convert Roman Numeral to Integer

### Constraint

```
1 < numberGenerated < 9999
```

### Example

```
VIII             =>   8
LXXIV            =>   74
C                =>   100
CI               =>   101
CXXIII           =>   123
MIV              =>   1004
MMMMDCCCXXI      =>   4821
MMMMXLV          =>   4045
MMMMMC           =>   5100
MMMMMMMMMCMXCIX  =>   9999
```
